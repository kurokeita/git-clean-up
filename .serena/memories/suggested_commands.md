# Suggested commands

- `pnpm dev` run the CLI in development mode.
- `pnpm test` run Vitest test suite.
- `pnpm run check` run Biome, markdownlint, and TypeScript checks.
- `pnpm run build` create the distributable CLI in `dist/`.
- `pnpm run format` apply Biome formatting and markdownlint fixes.
- `git status --short` inspect working tree state.
- `rg --files` list repository files quickly.
- `rg "pattern" src` search source code.
