# Tech Stack

## Core

- **Primary Language**: TypeScript (v5.x+)

- **Runtime**: Node.js (v18.x+)

- **Package Manager**: `pnpm`

## CLI Framework & UI

- **Command Parsing**: `commander`

- **Interactive Prompts**: `@clack/prompts`

- **Terminal Styling**: `picocolors`

## Git & Execution

- **Git Integration**: `simple-git`

- **Process Execution**: `execa`

## Development & Testing

- **Testing Framework**: `vitest`

- **Bundler**: `tsup` (for zero-config TypeScript bundling)

- **Linting**: `eslint`

- **Formatting**: `prettier`

- **Type Checking**: `tsc`

## Key Dependencies

- `commander`: Command-line interface builder.

- `@clack/prompts`: Beautiful, accessible CLI prompts.

- `picocolors`: Fastest and smallest library for terminal text styling.

- `simple-git`: A light weight interface for running git commands in any node.js application.

- `execa`: Process execution for node.js.
